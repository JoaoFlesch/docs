## O que são Projetos

Projetos são ambientes independentes dentro de um Time.

Cada projeto possui:

- Domínios
- Páginas
- Links
- Pixels
- Campanhas

---

## Organização recomendada

```text
1 Produto = 1 Projeto
```

Isso facilita:

- Relatórios
- Integrações
- Rastreamento
