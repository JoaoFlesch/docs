## O que é o TrackUp?

TrackUp é uma plataforma all-in-one para profissionais de tráfego pago.

Com o TrackUp você centraliza:

- Rastreamento
- Páginas
- Links
- Eventos
- Campanhas
- Integrações

Tudo em uma única plataforma.

<Tip>
  ## Como o TrackUp funciona

  O fluxo básico é:

  Anúncio

   ↓

  Link do TrackUp

   ↓

  Página

   ↓

  Checkout

   ↓

  Conversão

   ↓

  Pixels e Relatórios
</Tip>

Durante esse processo o TrackUp captura:

- UTMs
- Click IDs
- Fingerprint
- Dados do dispositivo
- Dados de navegação
- Conversões
